# 🌍 Universal AI Translator (200+ Languages)

An AI-powered multilingual translation application supporting 200+ languages using Hugging Face Transformer models.

---

## 🚀 Features

- Translate across 200+ languages
- Real-time translation
- AI-powered language detection
- Interactive Gradio/Streamlit UI
- Fast transformer inference
- Google Colab compatible

---

## 🛠 Tech Stack

- Python
- Hugging Face Transformers
- NLLB-200
- NLP
- Gradio
- Streamlit

---

```bash
pip install -r requirements.txt
```
